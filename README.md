# 🧠 NEUROSCAN AI — Brain Tumor Detection Dashboard

NEUROSCAN AI is an interactive web-based dashboard for brain tumor analysis using MRI scans.

## 🚀 Features
- MRI scan upload
- AI tumor detection simulation
- Tumor segmentation visualization
- Clinical report generation
- AI chatbot assistant
- Responsive dashboard UI

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 🎯 Purpose
This project demonstrates how AI can assist in medical imaging and brain tumor diagnosis workflows.

## ⚠ Disclaimer
This project is only for educational/demo purposes and not for real medical diagnosis.
